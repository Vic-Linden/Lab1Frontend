# Portfolio Website

This is my personal portfolio website created as a school assignment.  
The purpose of the project was to build a responsive multi-page website using HTML, CSS, and vanilla JavaScript.

🌐 **Live demo:** [View the website](https://vic-linden.github.io/Lab1Frontend/)

## Features

- Responsive design for desktop, tablet, and mobile
- Interactive Easter eggs using vanilla JavaScript
  - Confetti animation triggered by Spacebar.
  - Clickable h2 headings with temporary text effect.

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid, media queries)
- Vanilla JavaScript
- Responsive Design with Media Queries
